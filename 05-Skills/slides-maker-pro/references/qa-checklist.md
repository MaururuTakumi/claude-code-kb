# QA Checklist

Run before finalizing any deck.

## Critical (must pass)
- [ ] No text clipping or overflow
- [ ] Action-title exists on each content slide
- [ ] Visual hierarchy is clear (title > key message > detail)
- [ ] Chart/table labels are readable
- [ ] Color consistency follows design tokens
- [ ] Source lines exist on factual data slides
- [ ] Page numbers are consistent
- [ ] File is editable `.pptx` (not image-only)

## Quality score (0-10 each)
1. Message clarity
2. Decision usefulness
3. Visual readability
4. Narrative flow
5. Consistency

Interpretation:
- 8.5+ Ready for executive use
- 7.5–8.4 Needs polish
- <7.5 Rework required

## High-risk failure patterns
- “Topic label” titles without conclusions
- Overloaded slides with tiny text
- Decorative charts without decision value
- Inconsistent spacing and alignment
- Unmarked assumptions presented as facts

## Fix order
1. Overflow/readability
2. Message-title clarity
3. Visual simplification
4. Alignment and spacing
5. Color and typography consistency
