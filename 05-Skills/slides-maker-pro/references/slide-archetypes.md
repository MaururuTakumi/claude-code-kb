# Slide Archetypes

Pick one archetype first, then adapt.

## A. Executive update (default)
1. Title
2. Executive summary
3. KPI snapshot
4. Trend deep dive
5. Issue analysis
6. Option comparison
7. Recommendation + roadmap
8. Risk / ask

## B. Proposal deck
1. Title / proposal headline
2. Client context and pain
3. Current gap
4. Solution architecture
5. Delivery plan
6. Team / operating model
7. Cost-benefit / ROI
8. Next step and decision request

## C. Strategy analysis
1. Strategic question
2. Executive answer
3. Market context
4. Competitive map (2x2)
5. Internal capability gap
6. Strategic options
7. Chosen strategy and rationale
8. 90-day action plan

## D. Product review
1. Product objective
2. User/problem summary
3. Usage funnel
4. Cohort or retention view
5. Key bottlenecks
6. Experiment options
7. Prioritization matrix
8. Release plan + KPI impact

## E. Training / education
1. Learning objective
2. Why it matters
3. Core concept 1
4. Core concept 2
5. Practical workflow
6. Case example
7. Common mistakes
8. Action checklist

## Visual mapping by slide intent
- Trend → line/column chart
- Share / composition → stacked bar or donut (if <=5 categories)
- Exact comparison → table
- Strategic positioning → 2x2
- Process / rollout → timeline or step flow
- Decision request → option matrix + recommendation block
